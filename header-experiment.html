Implementation Examples

JavaScript Implementation:

// Function to update header based on current city function updateHeaderForCity(cityKey) { const cityConfig = getCityConfig(cityKey); const logoElement = document.querySelector('.logo h1'); if (cityConfig) { // Update the logo text logoElement.innerHTML = `üêª Chunky ${cityConfig.name}`; // Update page title document.title = `Chunky ${cityConfig.name} - Bear Guide`; // Add city-specific styling if needed logoElement.classList.add('city-specific'); } else { // Fallback to default logoElement.innerHTML = 'üêª Chunky Dad'; document.title = 'Chunky Dad - Your Gay Bear Travel Guide'; logoElement.classList.remove('city-specific'); } } // Call this function when city changes // updateHeaderForCity('seattle'); // Example usage

CSS for City-Specific Styling:

/* Optional: Add city-specific styling */ .logo h1.city-specific { background: linear-gradient(45deg, #ff6b6b, #4ecdc4); -webkit-background-clip: text; -webkit-text-fill-color: transparent; background-clip: text; transition: all 0.3s ease; } /* City-specific color themes */ .logo h1[data-city="seattle"] { color: #2c5530; /* Seattle green */ } .logo h1[data-city="new-york"] { color: #0033cc; /* NYC blue */ }

Option 1: Full Implementation

Implement "Chunky [City]" on all city-specific pages:

  • Maintain consistent branding elsewhere

Best for: Balanced approach with clear hierarchy
