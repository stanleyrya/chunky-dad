Hyphenation Fix Test

Testing the new hyphenation logic for event names with shortName support

Test 1: ShortName with Optional Hyphens

These events have shortNames with "-" as optional hyphenation points. If the text fits, hyphens should be removed. Otherwise, they're used as breakpoints.

Test 2: ShortName with Escaped Hyphens

These events have shortNames with "\-" which are literal hyphens that must always be kept.

Test 3: Regular Names (No ShortName)

These events only have regular names. Hyphens in regular names should always be kept literally.

Test 4: Mixed Cases

Testing various edge cases and combinations.
