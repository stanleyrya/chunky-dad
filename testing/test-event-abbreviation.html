Event Abbreviation Test - Per Line Character Limits

Character Limits Per Line:

  • XS (< 375px): 5 characters per line
  • SM (375-768px): 8 characters per line
  • MD (768-1024px): 12 characters per line
  • LG (> 1024px): 20 characters per line

Note: The logic wraps text to multiple lines (max 3 lines for all breakpoints) based on these per-line limits. CSS word-wrap is set to 'normal' for standard wrapping behavior.
