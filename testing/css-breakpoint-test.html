üé® CSS-Based Breakpoint System

This demonstrates the new CSS-based approach where we generate all breakpoint versions and let CSS show/hide the appropriate one.

üì± How It Works

Implementation:

  1. JavaScript generates 4 versions of each event name (xs, sm, md, lg)
  2. Each version is optimized for its character limit
  3. CSS media queries show only the appropriate version
  4. No JavaScript width detection needed!

Benefits:

  • ‚úÖ Pure CSS responsiveness - no JavaScript calculations
  • ‚úÖ Instant response to window resizing
  • ‚úÖ Better performance (no DOM measurements)
  • ‚úÖ Cleaner separation of concerns
  • ‚úÖ Works with CSS container queries (future-proof)

üíª Code Example

JavaScript (in dynamic-calendar-loader.js):

generateEventNameElements(event) {
    const fullName = event.name || '';
    const hasShortName = !!(event.shortName || event.nickname);
    
    if (!hasShortName) {
        return `<div class="event-name">${fullName}</div>`;
    }
    
    // Generate different versions for each breakpoint
    const xsName = this.getSmartEventNameForBreakpoint(event, 'xs');
    const smName = this.getSmartEventNameForBreakpoint(event, 'sm');
    const mdName = this.getSmartEventNameForBreakpoint(event, 'md');
    const lgName = this.getSmartEventNameForBreakpoint(event, 'lg');
    
    return `
        <div class="event-name event-name-xs">${xsName}</div>
        <div class="event-name event-name-sm">${smName}</div>
        <div class="event-name event-name-md">${mdName}</div>
        <div class="event-name event-name-lg">${lgName}</div>
    `;
}

CSS (in styles.css):

/* Hide all breakpoint versions by default */
.event-name-xs, .event-name-sm, .event-name-md, .event-name-lg {
    display: none;
}

/* Show appropriate version based on screen size */
@media (max-width: 374px) {
    .event-name-xs { display: block; }
}

@media (min-width: 375px) and (max-width: 767px) {
    .event-name-sm { display: block; }
}

@media (min-width: 768px) and (max-width: 1023px) {
    .event-name-md { display: block; }
}

@media (min-width: 1024px) {
    .event-name-lg { display: block; }
}
