📅 Calendar Parsing Test & Debug Center

🔍 About This Test Page

Purpose: This comprehensive test page validates the calendar parsing functionality, logging system, and data processing capabilities of the Chunky Dad calendar system.

📋 What This Page Tests:

  • iCal File Parsing: Validates proper parsing of Google Calendar .ics files
  • Event Data Extraction: Tests extraction of event names, dates, times, locations, and descriptions
  • Recurring Event Processing: Validates weekly, monthly, and custom recurring patterns
  • Coordinate Parsing: Tests latitude/longitude extraction from location data
  • Description Field Parsing: Validates key-value pair extraction from event descriptions
  • Logging System: Comprehensive test of the centralized logging system with color-coded output
  • Error Handling: Tests graceful handling of malformed calendar data

📊 Sample Data Contains:

  • 7 Total Events with varying complexity levels
  • Weekly Events: Bears Are Animals, Bear Happy Hour, Beer Blast
  • Monthly Events: Goldiloxx, Last Saturday Party, Third Thursday Meetup, First Sunday Brunch
  • Complex Recurring Patterns: Last Saturday, Third Thursday, First Sunday
  • Location Data: Both coordinate-based and text-based locations
  • Rich Descriptions: Events with websites, Instagram, Google Maps links
  • Variable Cover Charges: Free, paid, and time-based pricing
